Technologies: JAVA8, SpringBoot, H2, Swagger, Groovy

1. Run application
In order to run application double click on buildAndRunApp.bat file. 

2. Open your browser and go to "http://localhost:8081/" .
You will see Swagger main page with all api which is currently available under "Users" group.
Click on api you want to use and you will see all information like required parameters and example response values.

3. Provide required parameters and click on "Try it out" to test API.

4. In order to close app, go to Command Prompt(from buildAndRunApp.bat), type ctrl+c, then confirm with "y" and press Enter.

5. All api-docs can be found in api-docs.yaml file
